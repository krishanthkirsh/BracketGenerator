# BracketGenerator

### What is this repository for? ###

* This is the console application was developed for an assgignment which is conducted by Amused group
* Version 1.0

### How do I get set up? ###
* Clone the project repostory to you local(git clone https://github.com/krishanthkirsh/BracketGenerator.git)
* Rebuild the project.


This Application Support Single Elimination and Group Stage tournament formats. Based on the user’s choice in the console, the application creates either format using a Factory Method design pattern.
